# miscellany
various scripts accumulated over time ?

- convert_skins.ps1: powershell script that takes a minecraft launcher "launcher_custom_skins.json" file and converts it to a format used by prism launcher and maybe some other launchers that use its specific format. also exports PNGs from the json file, which are necessary; if the manage skins window is opened and there is a skin entry that doesn't have a matching local image, it's removed.
  - you'll probably want to run this in a blank directory. it'll write to an index.json, and as many PNGs for each entry, with filenames dictated by the name property.
- Auto DPI.user.js: userscript (written for firefox with tampermonkey, idk how well it'll work in other environments) that sets a zoom level if the window width is wide enough. made because the mixed&fractional DPI implementation in wayland messes up things a little, and it's not really possible to make images look pixel perfect when your monitor scale is set to 125%. so instead i set monitor scale to 100% and let the browser scale pages instead. doesn't work that well probably. if a page loads fast enough then it'll flash for a moment before applying the zoom. if zoom is set below 100%, it temporarily disables automatic DPI letting you view the page at a 100% scale
